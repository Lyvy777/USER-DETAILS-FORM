# User Details Web Form

## Description
This project creates a simple web form using HTML and JavaScript that collects user details including:

- Full Name  
- Email Address  
- Phone Number  
- Gender (Male/Female/Other)  
- Comments  

The project demonstrates basic frontend web development concepts such as form design, validation, and simulated database submission.

### Form Design
- HTML is used to design a user-friendly form interface.
- Input fields are provided for:
  - Full Name
  - Email Address
  - Phone Number
  - Gender selection using radio buttons
  - Comments using a textarea

### JavaScript Validation
JavaScript is used to:

- Validate that all required fields are filled.
- Ensure the email field contains a valid email format using a regular expression.
- Display an alert message if any field is invalid.

### Form Submission (Simulation)
On successful validation, the form data is submitted using the **POST method**.

Database submission is simulated using JavaScript by:

- Sending the data to a mock URL. 
- Using the **fetch()** function.
- Displaying a success message on the web page:

Form submitted successfully!

## Technologies Used
- HTML – Form structure  
- CSS – Basic styling  
- JavaScript – Validation and submission logic  

## How to Run the Project
1. Download the project file.
2. Open:
   index.html
3. Fill in the form fields.
4. Click **Submit** to test validation and simulated submission.

## Learning Objectives
This project demonstrates:

- Form creation using HTML  
- Client-side validation using JavaScript  
- Email format validation  
- Using fetch() for HTTP requests  
- DOM manipulation  
- Basic frontend project structure
